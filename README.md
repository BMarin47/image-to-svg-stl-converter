# PNG to SVG to STL Converter

Herramienta en Python para convertir imágenes PNG en archivos SVG vectorizados y luego exportarlos automáticamente a STL para impresión 3D.

---

## Funciones

- Carga de imágenes PNG  
- Conversión a SVG  
- Conversión automática a STL  
- Exportación de archivos listos para slicer  

---

## Uso

1. Colocar PNG en `input/`
2. Ejecutar `app.py`
3. Buscar resultados en `output/`

---

## Requisitos

- Python 3  
- pip  
- dependencias de `requirements.txt`  

---

## Ejemplo

Entrada:
- imagen PNG  

Salida:
- archivo SVG  
- archivo STL listo para impresión  

---

python app.py
